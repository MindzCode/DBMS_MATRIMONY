# Matrimonial Website Project in Node.js and MySQL

## Project Overview
This is an online matrimonial website built using Node.js and MySQL. The platform allows users to register, create profiles, set partner preferences, browse other profiles, and connect with potential matches.

## Features
- User Registration and Login
- Enter Personal Details and Family Details
- Set Partner Preferences
- Dashboard to view profiles of opposite gender
- "My Matches" section based on partner preferences
- Send invitations to other users and exchange messages upon acceptance
- Multiple search options:
  - Simple Search
  - Advanced Search
  - Search by User ID
- Interact with other members:
  - Like profiles
  - Follow members

## Technology Stack
- Node.js: Backend runtime environment
- MySQL: Database to store user data
- CSS: Styling and layout
- Bootstrap: Responsive design
- JavaScript: Client-side scripting
- jQuery: DOM manipulation and AJAX
